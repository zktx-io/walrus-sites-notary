# Contributing

Thank you for considering contributing to Notary for Walrus Sites & MVR Contracts.

## How to Contribute

- Fork the repository  
- Create a feature branch  
- Submit a pull request against `main`  

## Contribution Requirements

All contributions must:

- Pass CI checks before merging  
- Follow the existing project structure and coding style  
- Include clear and descriptive commit messages  
- Not introduce secrets or sensitive information  
- Be reviewed and approved by a maintainer before merge  

### Testing Requirements

All new functionality must include corresponding automated tests where applicable.

- Tests must pass locally and in CI  
- Changes that affect verification logic should include coverage for expected success and failure cases  

All changes to `main` require pull requests.
